Questions, answered

Good to know.

Does ClaudeMeter collect analytics?

The native application collects no telemetry. This public project website uses only an anonymous Hits counter that counts page requests without identifying unique visitors.

Do I need an API key?

No. ClaudeMeter reads your existing Claude Code credentials and local Codex logs. It does not ask you to create another provider key.

Why is the app so lightweight?

It is written in Rust and uses native Windows and macOS interfaces instead of bundling Chromium, Node.js, Python or .NET.

Which systems are supported?

Windows 10/11 and Apple Silicon Macs running macOS 12 or newer are supported by the current releases.
